# k = 0 - failure (tails)
# k = 1 - success (heads)
def bernoulli_model(k, theta, prior):
    posterior = theta**k * (1 - theta)**(1 - k) * prior
    return normalise(posterior, theta)

After the first update the distribution has is zero for

θ¶

0 . Why? We have observed one success so we know immediately that there is a non-zero success probability. After the second update the distribution is zero at

θ¶

1 . Why? We have observed one failures, so we know that success is not certain.

Successive updates result in progressively narrower distributions. The distribution that emerges is a Beta distribution!

The observations can be presented to the model in any order and still achieve the same result. Equally, the observations can all be processed at once by formulating their joint likelihood.
